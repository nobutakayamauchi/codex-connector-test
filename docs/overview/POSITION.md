# POSITION

This repository exists to validate a narrow pre-test workflow for Codex:

1. Read repository instructions (including `AGENTS.md`).
2. Read a minimal local registry (`registry/talent-registry.yaml`).
3. Execute sandbox test tasks under `tasks/`.
4. Produce local result artifacts under `results/` and/or `rts-records/`.
5. Report in RTS-style sections: facts / assumptions / unverified / risks.

## Non-goals
- Hosting production RTS logic.
- Embedding Skills, MCP Packs, Hermes, or Signal Feed implementations.
- Performing external side effects or environment mutation.
